# 🚀 Startup Discovery Landing Page

## 📌 Project Description
This is a responsive landing page built using HTML and CSS.  
It is designed as a Startup Discovery Platform where users can explore, connect, and build startups.

## 🛠 Technologies Used
- HTML5
- CSS3
- Flexbox
- Media Queries

## ✨ Features
- Responsive design
- Modern UI with animations
- Hero section with CTA
- Startup showcase cards
- Clean and minimal layout
